# Perps - 首页底部状态

> **模块**：Perps 合约交易  
> **App 版本**：待定  
> **日期**：2026-03-05  
> **测试端**：桌面端 / 插件 / Web 大屏（不包含移动端小屏）

---

## 需求背景

- 收藏代币后，Perps 首页顶部展示收藏代币，支持 $ / % 切换、代币 icon，**新增拖动排序**。
- 页面底部新增**连接状态检测**、速率显示、刷新按钮、设置（底部轮播选项）、合约社区与「更多」入口。

---

## 功能描述

### 1. 首页顶部收藏代币（在原有收藏功能上增强）

- 展示：收藏代币横向排列，带代币 icon，支持 $（涨跌额）与 %（涨跌幅）切换。
- **新增**：支持拖动排序，排序结果持久化并与自选列表、底部轮播（当选择「收藏」时）一致。

### 2. 页面底部连接状态

| 子项 | 说明 |
|------|------|
| 状态显示 | 连接正常 / 连接失败；正常=绿色，失败=红色 |
| 速率显示 | 示例：「连接正常 279ms」 |
| 刷新 | 刷新按钮，点击刷新接口数据 |
| 设置 | 齿轮按钮，控制页面底部代币轮播展示方式 |

### 3. 底部轮播设置（设置弹窗内）

- **不展示**：不展示底部代币轮播。
- **默认数据**：展示默认代币轮播（如 Popular）。**根据当前模式联动**：合约模式下展示合约热门代币，现货模式下展示现货热门交易对；模式切换时轮播内容自动切换。
- **收藏代币**：展示收藏代币轮播，与顶部收藏列表及排序一致。
- **点击切换**：底部轮播中点击某一代币的切换逻辑与顶部「收藏」一致——当前选中的代币切换为该代币，图表与数据（如盘口、仓位等）切换为对应代币。

### 4. 底部链接

- **合约社区**：点击跳转 TG 频道。
- **更多**：下拉包含「帮助」「查看教程」「用户协议」「隐私政策」，点击跳转对应文章/页面。

---

## 业务规则摘要

- 仅桌面端 / 插件 / Web 大屏生效；移动端可不展示或逻辑可差异。
- 顶部收藏与底部「收藏」轮播：数据与排序完全同步；底部轮播点击某代币后，图表与数据切换逻辑与顶部收藏条点击一致。
- 连接状态依赖实际接口/WS 健康检测；失败时显示红色与「连接失败」。
- 更多菜单：关于、帮助、查看教程、用户协议、隐私政策（以实际产品文案为准）。

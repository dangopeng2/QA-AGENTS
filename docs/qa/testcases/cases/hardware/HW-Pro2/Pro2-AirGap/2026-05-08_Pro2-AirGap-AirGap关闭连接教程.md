# Pro2 - 连接 App 钱包入口教程（AirGap 关闭对照组）

> 规则文档：`docs/qa/rules/hardware-rules.md`
> 测试端：OneKey App（iOS / Android / Desktop / Extension / Web）
> 关联：作为 Pro2-AirGap 系列对照组（与「AirGap 开启连接禁用」用例对照）
> 变更说明：合并原「蓝牙教程」「USB 教程」「二维码连接」三段子流程，统一在 AirGap 关闭场景下覆盖
>
> **术语约定**：本文「**跳转引导码**」专指教程页中给用户**手机扫描**的官网链接二维码，与「AirGap 协议二维码（Pro2 协议层）」**不是同一概念**。

---

## 前置条件与测试数据

1. App 已创建/导入钱包
2. Pro2 设备已**关闭「AirGap」开关**
3. 进入 App 端「连接 App 钱包」入口（在 DApp / 外部钱包连接流程中触发）
4. 网络可达，跳转引导码扫描后可正常跳转到 onekey.so 教程页

> **关于「OneKey Wallet」选项**：本入口出现的「OneKey Wallet」指的是**连接 OneKey App 客户端**（即 OneKey 自家 App，作为可签名客户端使用）。

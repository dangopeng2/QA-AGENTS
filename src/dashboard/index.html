⚙ 账户配置

仅用于转账（Transfer）类用例。脚本默认从「账户1」转账到「账户2」；当账户1余额不足时，脚本会自动反向（账户2 发送、账户1 接收）。地址不在此填写——不同链格式不同，脚本会按当前链自动从所选账户中读取。其他类型的用例（Swap / DeFi 等）后续会有各自的配置入口。

每个人在自己机器上配一次即可（shared/runtime-config.json 已在 .gitignore，不会入库）。
